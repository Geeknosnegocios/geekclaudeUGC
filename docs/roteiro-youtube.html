01

Clonar o repositório

Baixa o código do GitHub pro seu PC.

Abre o terminal (PowerShell no Windows, Terminal no Mac/Linux) e roda:

git clone https://github.com/Geeknosnegocios/geekclaudeUGC.git
cd geekclaudeUGC
ℹ️
Não tem git? Windows winget install Git.Git · Mac brew install git · Linux sudo apt install git
02

Instalar Python 3.10+

Linguagem que roda o build pipeline.

SOComo instalar
Windowspython.org/downloads/windows — marca "Add Python to PATH"
Macbrew install python@3.11
Linuxsudo apt install python3 python3-pip

Confere que instalou:

python --version

Deve aparecer Python 3.10 ou superior.

03

Instalar dependências Python

Bibliotecas que o build usa: requests, dotenv, ElevenLabs, faster-whisper, imageio-ffmpeg.

Dentro da pasta geekclaudeUGC:

pip install -r requirements.txt
ℹ️
Demora 1-2 min na primeira vez. O faster-whisper baixa modelos pequenos do Whisper local.
04

Instalar ffmpeg

Ferramenta de processamento de vídeo. O coração do pipeline.

SOComando
Windowswinget install ffmpeg ou scoop install ffmpeg
Macbrew install ffmpeg
Linuxsudo apt install ffmpeg
💡
Não conseguiu? Tranquilo. O pacote imageio-ffmpeg (já veio no Passo 3) tem ffmpeg embutido e vai funcionar igual.
05

Instalar Bun

Runtime JavaScript que roda o HyperFrames (camada de polish profissional).

Windows (PowerShell):

powershell -c "irm bun.sh/install.ps1 | iex"

Mac / Linux:

curl -fsSL https://bun.sh/install | bash

Reabre o terminal e confere:

bun --version
06

Instalar HyperFrames

Camada de polish premium. 85 blocks profissionais grátis (kinetic captions, transitions shader, lower-thirds, end CTA, vignette, grain).

Ainda na pasta geekclaudeUGC:

mkdir -p vendor
git clone --depth=1 https://github.com/heygen-com/hyperframes.git vendor/hyperframes
cd vendor/hyperframes
bun install
bun run build
cd ../..
⚠️
A instalação baixa Chromium (~170 MB). Leva 2-3 minutos na primeira vez. Roda só 1× na vida do projeto.

Confere se funcionou:

bin/hf doctor

Tem que mostrar ✓ Version, ✓ Node.js, ✓ FFmpeg, ✓ Chrome.

07

Criar contas + pegar chaves API

5 serviços, todos grátis. Total: ~5 minutos.

🎥 Pexels — b-roll grátis ∞

  1. Acessa pexels.com/api
  2. Cria conta
  3. Copia a Authorization key

🖼️ Pixabay — imagens + vídeos grátis

  1. Cria conta em pixabay.com/accounts/register
  2. Após login, Profile → tab API
  3. Copia a chave

🎤 ElevenLabs — TTS 10k chars/mês grátis

  1. Cria conta em elevenlabs.io
  2. Profile → API Key
  3. Copia

🎵 Jamendo — música grátis (35k tracks)

  1. Cria conta em devportal.jamendo.com
  2. My Apps → Create new app
  3. Nome qualquer (ex: geekclaudeUGC)
  4. Copia o Client ID

08

Preencher o .env

Cola as chaves no arquivo de configuração.

Duplica o template:

cp .env.example .env

Abre .env no seu editor (VSCode, Notepad++) e preenche:

PEXELS_KEY=cola_sua_chave_aqui
PIXABAY_KEY=cola_sua_chave_aqui
ELEVEN_KEY=cola_sua_chave_aqui
ELEVEN_VOICE_ID=cgSgspJ2msm6clMCkdW9
JAMENDO_CLIENT_ID=cola_seu_client_id_aqui
ℹ️
Voz padrão cgSgspJ2msm6clMCkdW9 = Jessica multilingual (free OK). Pra voz PT-BR nativa Larissa: OjcGK1RXdMD1PFj2eIuN (só funciona com ElevenLabs Starter $5/mo ou rateio).
⚠️
Nunca commite o .env pro GitHub. Já está no .gitignore.

Baixar o MP4

Creative Center não tem botão direto. Métodos:

  • Win+G (Xbox Game Bar no Windows)

11

Pedir clone pro agente

O agente faz o trabalho duro: transcreve, analisa beats, reescreve script PT-BR.

No chat do Claude/Codex/Gemini, cola este prompt:

Clona esse ad pra mim. É de [seu nicho].
Vou anunciar meu produto [nome do seu produto].
Aqui está a foto do produto: [anexa imagem]
Referência salva em: 01_talking_head/assets/reference.mp4

O que o agente vai entregar

  1. Análise: setting, character, beats da referência
  2. Script PT-BR adaptado pro seu produto
  3. Briefing HeyGen: avatar, voz, cenário
  4. Prompts SuperGrok: 3-6 prompts pra gerar clips
  5. Asset checklist: caminhos exatos pra salvar cada arquivo
  6. Comando final pra rodar o build
ℹ️
O agente automaticamente escolhe o template certo (01 talking_head / 02 unboxing / 03 faceless / 04 app_promo / 05 extend) baseado na referência.

12

Gerar avatar no HeyGen

Só se o template usa avatar (templates 01, 02, 04).

  1. Acessa app.heygen.com
  2. Clica em Create Video → AI Avatar (ou Avatar IV)
  3. Cola o script que o agente te deu
  4. Escolhe avatar sugerido no briefing
  5. Escolhe voz PT-BR sugerida
  6. Aspect: 9:16 vertical
  7. Resolução: 720p ou superior
  8. Clica Generate (1-3 min)
  9. Download MP4
  10. Salva no caminho que o agente indicou (tipo: 01_talking_head/assets/heygen.mp4)
⚠️
Se template é 02 (unboxing), você gera 2 vídeos: heygen_intro.mp4 e heygen_outro.mp4.

13

Gerar clips no SuperGrok

Só se template usa clips de produto/cena (templates 02 e 03).

  1. Acessa grok.com (precisa SuperGrok pago ou rateio)
  2. Clica em Imagine
  3. Cola o primeiro prompt do agente
  4. Espera gerar imagem (~30-60s)
  5. Clica Animate / Make Video
  6. Aspect: 9:16 vertical ← não esquece
  7. Duração: 6 segundos
  8. Download MP4
  9. Salva em 03_faceless_lifestyle/assets/clips/clip_01.mp4
  10. Repete pros prompts 2, 3, 4 (vira clip_02.mp4, clip_03.mp4, clip_04.mp4)
⚠️
Crítico: se aspect ratio do Grok não estiver em 9:16 explícito, vídeo sai 1:1 quadrado e o pipeline adiciona faixas pretas. Confere antes de baixar.
14

Rodar o build base

O agente executa por você. Você só assiste.

Fala "go" / "manda" / "ok" no chat. O agente roda:

python 03_faceless_lifestyle/build.py --tts "Seu script PT-BR aqui" --srt

O que acontece em ~30 segundos

  1. ElevenLabs gera voz feminina natural
  2. faster-whisper transcreve word-level pra legenda
  3. Jamendo baixa música de fundo automática
  4. ffmpeg normaliza os 4 clips
  5. Concatena tudo
  6. Mixa voiceover com música (música em -22 dB)
  7. Salva MP4 final + arquivo .srt

Output em:

03_faceless_lifestyle/outputs/lifestyle_v1.mp4
03_faceless_lifestyle/outputs/lifestyle_v1.srt
💡
Esse MP4 já tá pronto pra subir. Mas se quer polish profissional com captions kinéticos, transições, lower-third e CTA, segue pro Passo 15.
15

Polish com HyperFrames

A cereja do bolo. Adiciona kinetic captions, white-flash transitions, lower-third, end CTA card, shimmer, grain, vignette.

Comando 1 — Scaffold composition

bin/hf init compositions/meu_ad --no-install

Comando 2 — Copia assets

cp 03_faceless_lifestyle/assets/clips/*.mp4 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/voiceover.mp3 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/music.mp3 compositions/meu_ad/

Comando 3 — Adiciona blocks do catálogo

bin/hf add caption-highlight whip-pan grain-overlay vignette shimmer-sweep tiktok-follow

Comando 4 — Pede pro agente clonar a composition reference

Clona a composition compositions/lucrown_ad/index.html
para compositions/meu_ad/index.html.
Ajusta script, voiceover e produto pro meu ad.

Comando 5 — Lint + render

bin/hf lint
bin/hf render -o 03_faceless_lifestyle/outputs/meu_ad_hf.mp4

Render leva ~30 segundos. Sai um MP4 com polish nível agência.

💡
Quer mais opções? Roda bin/hf catalog list — tem 85 blocks: caption-particle-burst, caption-glitch-rgb, cinematic-zoom, vfx-iphone-device, apple-money-count, instagram-follow, e mais.
