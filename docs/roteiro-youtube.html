FASE 01 / 04

SETUP BASE

Instala as ferramentas base no seu PC. Python, ffmpeg, Bun, HyperFrames. Roda 1x na vida.

[01]

Clonar o repositório

Baixa o código do GitHub.

git clone https://github.com/Geeknosnegocios/geekclaudeUGC.git
cd geekclaudeUGC
[i]
Sem git? winget install Git.Git · brew install git · sudo apt install git
[02]

Instalar Python 3.10+

Linguagem que roda o build pipeline.

SOComo
Windowspython.org + marcar "Add to PATH"
Macbrew install python@3.11
Linuxsudo apt install python3 python3-pip
 
python --version
[03]

Dependências Python

Bibliotecas que o build usa.

pip install -r requirements.txt
[i]
Demora 1-2 min. faster-whisper baixa modelos do Whisper.
[04]

Instalar ffmpeg

Processamento de vídeo. Coração do pipeline.

SOComando
Windowswinget install ffmpeg
Macbrew install ffmpeg
Linuxsudo apt install ffmpeg
[ok]
Não rolou? Tranquilo. imageio-ffmpeg (Passo 03) traz embutido.
[05]

Instalar Bun

Runtime JS que roda o HyperFrames (camada polish).

Windows

powershell -c "irm bun.sh/install.ps1 | iex"

Mac / Linux

curl -fsSL https://bun.sh/install | bash
[06]

Instalar HyperFrames

Camada de polish premium. 85 blocks profissionais grátis.

mkdir -p vendor
git clone --depth=1 https://github.com/heygen-com/hyperframes.git vendor/hyperframes
cd vendor/hyperframes && bun install && bun run build
cd ../..
[!]
Baixa Chromium ~170MB. 2-3 min na primeira vez. Roda 1× na vida.
bin/hf doctor

[12]

Gerar avatar no HeyGen

Só se template usa avatar (01, 02, 04).

  1. Abre app.heygen.com
  2. Create Video → AI Avatar
  3. Cola o script
  4. Escolhe avatar + voz sugeridos
  5. Aspect 9:16, 720p+
  6. Generate (1-3 min) → Download MP4
  7. Salva em assets/heygen.mp4
[!]
Template 02 precisa de 2 vídeos: heygen_intro.mp4 + heygen_outro.mp4.
[13]

Gerar clips no SuperGrok

Só se template usa clips de cena (02, 03).

  1. Abre grok.com
  2. Imagine → cola prompt → gera imagem
  3. Animate → vídeo 6s
  4. Aspect 9:16 explícito
  5. Download → salva como clip_01.mp4, clip_02.mp4, ...
[!]
CRÍTICO: se aspect ratio não estiver 9:16, vídeo sai 1:1 e o pipeline adiciona faixas pretas.
FASE 04 / 04

FINALIZAR

Build base → polish HyperFrames → upload TikTok. Hora de subir o foguete.

[14]

Rodar o build base

Agente executa. Você só assiste.

python 03_faceless_lifestyle/build.py --tts "Seu script PT-BR" --srt

O que rola em 30s

  1. ElevenLabs gera a voz
  2. Whisper transcreve word-level
  3. Jamendo baixa música automática
  4. ffmpeg normaliza + concatena clips
  5. Mixa voiceover + música
  6. Salva MP4 + .srt
03_faceless_lifestyle/outputs/lifestyle_v1.mp4
03_faceless_lifestyle/outputs/lifestyle_v1.srt
[15]

Polish com HyperFrames

Kinetic captions, transitions shader, lower-third, end CTA, grain, vignette.

bin/hf init compositions/meu_ad --no-install
cp 03_faceless_lifestyle/assets/clips/*.mp4 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/voiceover.mp3 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/music.mp3 compositions/meu_ad/
bin/hf add caption-highlight whip-pan grain-overlay vignette shimmer-sweep tiktok-follow

Pede pro agente

Clona a composition compositions/lucrown_ad/index.html
pra compositions/meu_ad/index.html.
Ajusta script + voiceover + produto pro meu ad.
bin/hf render -o 03_faceless_lifestyle/outputs/meu_ad_hf.mp4
[ok]
