Geek CLAUDE - UGC

Tutorial passo a passo pra instalar e gerar seu primeiro ad UGC em 20 minutos. Sem editor. Sem aparecer. Sem gastar com produção.

Windows / Mac / Linux Python 3.10+ Node 22+ Open-source MIT

O que você vai fazer

  1. Clonar o repositório
  2. Instalar Python
  3. Instalar dependências Python
  4. Instalar ffmpeg
  5. Instalar Bun
  6. Instalar HyperFrames
  7. Criar contas + chaves API
  8. Preencher o .env
  9. Abrir o agente (Claude / Codex / Gemini)
  10. Achar ad referência no TikTok
  11. Pedir clone pro agente
  12. Gerar avatar no HeyGen
  13. Gerar clips no SuperGrok
  14. Rodar build base
  15. Polish final com HyperFrames
  16. Subir pro TikTok / Reels
1

Clonar o repositório

Baixa o código todo pro seu PC.

Abre o terminal (PowerShell no Windows, Terminal no Mac/Linux) e roda:

git clone https://github.com/Geeknosnegocios/geekclaudeUGC.git
cd geekclaudeUGC
Se não tem git instalado: Windows winget install Git.Git · Mac brew install git · Linux sudo apt install git
2

Instalar Python 3.10+

Linguagem que roda o build pipeline.

SOComo instalar
Windowspython.org/downloads/windows — marca "Add Python to PATH" durante instalação
Macbrew install python@3.11
Linuxsudo apt install python3 python3-pip

Confere que instalou:

python --version

Deve aparecer Python 3.10 ou superior.

3

Instalar dependências Python

Bibliotecas que o build usa: requests, dotenv, ElevenLabs, faster-whisper, imageio-ffmpeg.

Dentro da pasta geekclaudeUGC:

pip install -r requirements.txt
Demora 1-2 min na primeira vez. faster-whisper baixa modelos pequenos do Whisper.
4

Instalar ffmpeg

Ferramenta de processamento de vídeo. O coração do pipeline.

SOComando
Windowswinget install ffmpeg ou scoop install ffmpeg
Macbrew install ffmpeg
Linuxsudo apt install ffmpeg
Não conseguiu? Tranquilo. O pacote imageio-ffmpeg (já veio no Passo 3) tem um ffmpeg embutido que vai funcionar do mesmo jeito.
5

Instalar Bun

Runtime JavaScript que roda o HyperFrames (camada de polish).

Windows (PowerShell):

powershell -c "irm bun.sh/install.ps1 | iex"

Mac / Linux:

curl -fsSL https://bun.sh/install | bash

Reabre o terminal e confere:

bun --version
6

Instalar HyperFrames

A camada de polish premium. 85 blocks profissionais grátis (kinetic captions, transitions shader, lower-thirds, end CTA, vignette, grain).

Ainda na pasta geekclaudeUGC:

mkdir -p vendor
git clone --depth=1 https://github.com/heygen-com/hyperframes.git vendor/hyperframes
cd vendor/hyperframes
bun install
bun run build
cd ../..
A instalação baixa Chromium (~170 MB). Leva 2-3 minutos na primeira vez. Roda só 1x na vida.

Confere se funcionou:

bin/hf doctor

Tem que mostrar "✓ Version", "✓ Node.js", "✓ FFmpeg", "✓ Chrome".

7

Criar contas + pegar chaves API

5 serviços, todos grátis. Total: ~5 minutos.

🎥 Pexels (b-roll grátis ilimitado)

  1. Acessa pexels.com/api
  2. Cria conta
  3. Copia a Authorization key que aparece

🖼️ Pixabay (imagens + vídeos grátis)

  1. Cria conta em pixabay.com/accounts/register
  2. Após login, vai em Profile → tab API
  3. Copia a chave

🎤 ElevenLabs (TTS — 10k chars/mês grátis)

  1. Cria conta em elevenlabs.io
  2. Vai em Profile → API Key
  3. Copia

🎵 Jamendo (música grátis — 35k tracks)

  1. Cria conta em devportal.jamendo.com
  2. Vai em My Apps → Create new app
  3. Coloca nome qualquer (ex: geekclaudeUGC)
  4. Copia o Client ID
⚡ Bônus pago (opcional)

Preencher o .env

Cola as chaves no arquivo de configuração.

Duplica o template:

cp .env.example .env

Abre o .env no seu editor (VSCode, Notepad++, etc) e preenche:

PEXELS_KEY=cola_sua_chave_aqui
PIXABAY_KEY=cola_sua_chave_aqui
ELEVEN_KEY=cola_sua_chave_aqui
ELEVEN_VOICE_ID=cgSgspJ2msm6clMCkdW9
JAMENDO_CLIENT_ID=cola_seu_client_id_aqui
A voz cgSgspJ2msm6clMCkdW9 é a Jessica multilingual (funciona free). Pra voz PT-BR nativa Larissa, troca pra OjcGK1RXdMD1PFj2eIuN — só funciona com ElevenLabs plano Starter $5/mo (ou rateio).
Nunca commita o .env pro GitHub. Já tá no .gitignore.

O Creative Center não tem botão download direto. Métodos:

  • Win+G (Xbox Game Bar no Windows)

11

Pedir o clone pro agente

O agente faz o trabalho duro: transcreve, analisa beats, reescreve script PT-BR.

No terminal do Claude/Codex/Gemini, cola este prompt:

Clona esse ad pra mim. É de [seu nicho].
Vou anunciar meu produto [nome do seu produto].
Aqui está a foto do produto: [anexa imagem]
Referência salva em: 01_talking_head/assets/reference.mp4

O que o agente vai te entregar

  1. Análise da referência: setting, character, beats
  2. Script PT-BR adaptado pro seu produto
  3. Briefing HeyGen: qual avatar, qual voz, qual cenário
  4. Prompts SuperGrok: 3-6 prompts pra gerar os clips
  5. Asset checklist: caminhos exatos onde salvar cada arquivo
  6. Comando final pra rodar o build
O agente automaticamente escolhe o template certo (01 talking_head / 02 unboxing / 03 faceless / 04 app_promo / 05 extend) baseado na referência.

12

Gerar avatar no HeyGen

Só se o template escolhido usar avatar (templates 01, 02, 04).

  1. Acessa app.heygen.com
  2. Clica em Create Video → AI Avatar (ou Avatar IV)
  3. Cola o script que o agente te deu
  4. Escolhe o avatar sugerido no briefing
  5. Escolhe a voz PT-BR sugerida
  6. Aspect ratio: 9:16 vertical
  7. Resolução: 720p ou superior
  8. Clica Generate (espera 1-3 min)
  9. Download MP4
  10. Salva exatamente no caminho indicado pelo agente, tipo: 01_talking_head/assets/heygen.mp4
Se o template é 02 (unboxing), você gera 2 vídeos: um heygen_intro.mp4 e outro heygen_outro.mp4.

13

Gerar clips no SuperGrok

Só se o template usa clips de produto/cena (templates 02 e 03).

  1. Acessa grok.com (precisa SuperGrok pago ou rateio)
  2. Clica em Imagine
  3. Cola o primeiro prompt que o agente te deu
  4. Espera gerar a imagem (~30-60s)
  5. Clica em Animate / Make Video
  6. Aspect ratio: 9:16 vertical (não esquece!)
  7. Duração: 6 segundos
  8. Download MP4
  9. Salva em: 03_faceless_lifestyle/assets/clips/clip_01.mp4
  10. Repete pros prompts 2, 3, 4 (vira clip_02.mp4, clip_03.mp4, clip_04.mp4)
Atenção: se o aspect ratio não estiver em 9:16 explícito no Grok, o vídeo sai 1:1 quadrado e o pipeline adiciona faixas pretas no topo e fundo. Confere antes de baixar.
14

Rodar o build base

O agente executa por você. Você só assiste.

Volta no terminal do agente e fala "go" / "manda" / "ok". Ele vai rodar algo tipo:

python 03_faceless_lifestyle/build.py --tts "Seu script PT-BR aqui" --srt

O que acontece em ~30 segundos

  1. ElevenLabs gera voz feminina natural
  2. faster-whisper transcreve voz word-level pra legenda
  3. Jamendo baixa música de fundo automática (instrumental, sem voz)
  4. ffmpeg normaliza os 4 clips (corta, padroniza, normaliza áudio)
  5. Concatena tudo
  6. Mixa voiceover com música (música em -22 dB)
  7. Salva MP4 final + arquivo de legenda .srt

Saída no caminho:

03_faceless_lifestyle/outputs/lifestyle_v1.mp4
03_faceless_lifestyle/outputs/lifestyle_v1.srt
Esse MP4 já tá pronto pra subir. Mas se quer polish profissional com captions kinéticos, transições, lower-third e CTA, segue pro Passo 15.
15

Polish final com HyperFrames

A cereja do bolo. Adiciona kinetic captions, white-flash transitions, lower-third brand, end CTA card, shimmer, grain, vignette.

Comando 1 — Scaffold composition

bin/hf init compositions/meu_ad --no-install

Comando 2 — Copia assets pra dentro da composition

cp 03_faceless_lifestyle/assets/clips/*.mp4 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/voiceover.mp3 compositions/meu_ad/
cp 03_faceless_lifestyle/assets/music.mp3 compositions/meu_ad/

Comando 3 — Adiciona blocks do catálogo

bin/hf add caption-highlight whip-pan grain-overlay vignette shimmer-sweep tiktok-follow

Comando 4 — Pede pro agente clonar a composition reference

Cola no chat do agente:

Clona a composition compositions/lucrown_ad/index.html
para compositions/meu_ad/index.html.
Ajusta script, voiceover e produto pro meu ad.

Comando 5 — Lint + render

bin/hf lint
bin/hf render -o 03_faceless_lifestyle/outputs/meu_ad_hf.mp4

Render leva ~30 segundos. Sai um MP4 com polish nível agência.

Quer mais opções de polish? Roda bin/hf catalog list — tem 85 blocks: caption-particle-burst, caption-glitch-rgb, cinematic-zoom, vfx-iphone-device, apple-money-count, tiktok-follow, instagram-follow, e mais.
